        <div class="step-number">2</div>

        <p>启动开发服务器，实时预览修改效果</p>
        <div class="step-code"><pre><code>pnpm electron:dev</code></pre></div>
      </div>

        <h4>构建应用</h4>
        <p>编译打包，生成可分发的桌面应用</p>
        <div class="step-code"><pre><code>pnpm electron:build</code></pre></div>
